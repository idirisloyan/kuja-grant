SECTION 1: DATA SOURCES
================================================================================

--------------------------------------------------------------------------------
UN Security Council Consolidated List
--------------------------------------------------------------------------------
Download XML:  https://scsanctions.un.org/resources/xml/en/consolidated.xml
RSS Updates:   https://main.un.org/securitycouncil/en/rss-updates-unsc-consolidated-list
Format:        XML with DATAID, names, aliases, nationality, list type
API:           No official API; use XML download


--------------------------------------------------------------------------------
US OFAC SDN List
--------------------------------------------------------------------------------
Download CSV:  https://www.treasury.gov/ofac/downloads/sdn.csv
Addresses:     https://www.treasury.gov/ofac/downloads/add.csv
Aliases:       https://www.treasury.gov/ofac/downloads/alt.csv
Comments:      https://www.treasury.gov/ofac/downloads/sdn_comments.csv
Search Tool:   https://sanctionssearch.ofac.treas.gov/
Format:        CSV with ent_num (primary key), SDN_Name, SDN_Type, Program
API:           No official API; use CSV download

IMPORTANT:     Join all 4 files via ent_num for complete records


--------------------------------------------------------------------------------
EU Consolidated Financial Sanctions List
--------------------------------------------------------------------------------
Download CSV:  https://webgate.ec.europa.eu/fsd/fsf/public/files/csvFullSanctionsList_1_1/content?token=dG9rZW4tMjAxNw
Download XML:  https://webgate.ec.europa.eu/fsd/fsf/public/files/xmlFullSanctionsList_1_1/content?token=dG9rZW4tMjAxNw
RSS:           https://webgate.ec.europa.eu/fsd/fsf/public/rss
Search:        https://www.sanctionsmap.eu/
Token:         dG9rZW4tMjAxNw (base64 for "token-2017", public access)


--------------------------------------------------------------------------------
World Bank Debarment List
--------------------------------------------------------------------------------
Web:           https://www.worldbank.org/en/projects-operations/procurement/debarred-firms
Download:      No CSV/JSON download from WB; use OpenSanctions mirror
Updates:       Every 3 hours
Cross-debarment: AfDB, ADB, EBRD, IDB


--------------------------------------------------------------------------------
OpenSanctions (Unified API - Recommended for Implementation)
--------------------------------------------------------------------------------
API Base:      https://api.opensanctions.org/
Match:         POST https://api.opensanctions.org/match/<dataset>
Search:        GET  https://api.opensanctions.org/search/<dataset>?q=<query>
Entity:        GET  https://api.opensanctions.org/entities/<id>
Datasets:      un_sc_sanctions, us_ofac_sdn, eu_fsf,
               worldbank_debarred, sanctions (all combined)
Updated:       Daily
Auth:          API key required (free for non-commercial)
